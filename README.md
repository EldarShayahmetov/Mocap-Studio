# Mocap-Studio
Motion capture studio for using with 2 web-cams

# Basic Overview

The program allows you to calibrate the motion capture system and reconstruct the markers

![](Images/1.png)

## Markers

## Calibration

## Reconstruction

After calibration program reconstructs visible markers
